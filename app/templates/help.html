Help

You can view the GitHub Wiki, or, if you prefer, read a summary here.
This site has many basic features you'd expect from a chat client; here are some less obvious features:

Tagging

To tag a user, simply include their name in your message. The user who receives the message will see their tag with a bold border.

Private Messages

To pm a user, type "@username" followed by the message you want to send. PMs are not stored on the server and disappear, so careful about refresh if you wish to save a transcript! You can also pm a user by clicking on their name, which will autocomplete the "@username" in your bottom chat input.

Rooms

We're currently working on allowing multiple rooms in a single window (a single page application type deal), but for now, users start in the "lobby", and can move to a different chat room by directly modifying the url.

LaTeX

You can add a LaTeX to your message with standard methods.

Planned Developments

We're currently actively working on the chat section of this site, and plan on modifying it to accommodate questions as well.
